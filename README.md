# LLM Game-Theoretic Simulations

Two-player classical games with LLM agents.
Focus: how many communication rounds are needed to reach agreement/strategy.

Games:
- Prisoner's Dilemma
- Stag Hunt
- Battle of the Sexes
- Ultimatum Game

Modes:
- no_workflow
- workflow

Outputs:
- JSONL logs per episode in `data/runs/`
